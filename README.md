
# Проєкт Enrich

**Enrich** — це мобільний застосунок для підвищення ефективності вивчення англійської лексики. Основна ідея проєкту полягає у створенні персоналізованого навчального середовища через адаптивні інструменти.

---

##  Мета застосунку

Створення мобільної платформи для швидкого та регулярного вивчення англійських слів шляхом формування індивідуальних словників та проходження адаптивних тестів.

## Проблеми, які вирішує проєкт

* **Складність запам’ятовування:** допомога у вивченні нових слів та фраз під час самостійного навчання.
* **Відсутність практики:** запобігання швидкому забуванню матеріалу завдяки регулярним вправам.
* **Брак персоналізації:** розв'язання проблеми відсутності індивідуальних словників у більшості наявних рішень.
* **Розпорошеність ресурсів:** інтеграція всіх необхідних інструментів в єдину платформу.

##  Цільова аудиторія

* **Самоучки:** особи, які вивчають англійську мову самостійно.
* **Викладачі:** для розробки словників та контрольних тестів для своїх учнів.
* **Учні та студенти:** для підготовки до занять, іспитів та міжнародних тестів.

---

##  Основні функціональні вимоги

1. **Словник:** можливість створення власних карток, що включають слово, переклад та приклад вживання.
2. **Генератор тестів:** автоматичне формування квізів на основі доданих користувачем слів.
3. **Швидкий доступ:** функція перегляду слів або проходження міні-тестів без складного пошуку в інтерфейсі.

---

##  Технологічний стек (у розробці)

Для реалізації проєкту обрано сучасні інструменти, що забезпечують швидкість та кросплатформеність:

* **Frontend:** `React Native` — для створення зручного мобільного інтерфейсу, що працюватиме на iOS та Android.
* **Backend:** `FastAPI (Python)` — високоефективний фреймворк для обробки запитів, керування базою даних та логікою генерації тестів.

---

## Порівняльний аналіз

Згідно з аналізом ринку, **Enrich** має переваги над популярними конкурентами (*Quizlet, Duolingo, Lockbar*) у наступних аспектах:

* Повна можливість створення власного словника.
* Генерація тестів на основі **виключно** індивідуального контенту користувача.
* Повністю безкоштовний функціонал.
* Пріоритет на роботі саме з власними словами користувача.
